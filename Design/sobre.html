O Sistema

O Delibera é uma plataforma desenvolvida para facilitar a comunicação entre alunos e coordenação em instituições de ensino superior.

O sistema permite que alunos registrem solicitações formais de forma organizada e que coordenadores gerenciem essas demandas de maneira eficiente, centralizando todo o processo em uma única plataforma.

Funcionalidades Principais

  • Registro de solicitações por alunos (sem necessidade de cadastro)
  • Painel administrativo para coordenação
  • Gestão de múltiplas instituições
  • Sistema de planos individual e institucional
  • Notificações e acompanhamento de status
  • Relatórios e análises para gestão

Modelo de Negócio

O sistema opera no modelo SaaS (Software as a Service), oferecendo planos de assinatura mensal para coordenação e instituições de ensino.

Alunos utilizam o sistema gratuitamente, sem necessidade de cadastro ou pagamento.

Tecnologia

O sistema é desenvolvido com tecnologias modernas e seguras, incluindo:

  • Backend em Java com Spring Boot
  • Frontend com Thymeleaf, Bootstrap 5 e JavaScript
  • Banco de dados relacional
  • Hospedagem em nuvem com alta disponibilidade
  • Criptografia SSL/HTTPS

Conformidade Legal

O sistema está em conformidade com:

  • Lei Geral de Proteção de Dados (LGPD - Lei nº 13.709/2018)
  • Marco Civil da Internet (Lei nº 12.965/2014)
  • Código de Defesa do Consumidor
