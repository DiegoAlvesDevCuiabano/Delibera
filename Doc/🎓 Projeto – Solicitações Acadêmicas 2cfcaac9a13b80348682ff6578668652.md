# 🎓 Projeto – Solicitações Acadêmicas

# Visão Geral

**Problema:**
Processos acadêmicos são tratados via email, WhatsApp ou papel, sem histórico centralizado.

**Objetivo do MVP:**
Centralizar solicitações acadêmicas simples em um fluxo único e rastreável.

**Público-alvo:**
Alunos e coordenação acadêmic
